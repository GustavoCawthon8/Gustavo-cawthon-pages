O que é Html?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar a estrutura de páginas web. Ele organiza o conteúdo em elementos como títulos, parágrafos, imagens, links e tabelas, utilizando tags.

O que é Css?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a aparência e o design de páginas web criadas com HTML. Com o CSS, você pode personalizar cores, fontes, tamanhos, margens, alinhamento, e até criar animações.
